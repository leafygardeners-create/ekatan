# EKATAN

Premium residential interior design and execution ERP for the Indian market.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/core/` - Core modules (auth, db, config)
- `src/shared/` - Shared components, lib, types
- `src/features/` - Feature modules (self-contained)
- `src/app/` - Next.js App Router routes
